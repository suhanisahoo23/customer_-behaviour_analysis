# Customer Behaviour Analysis

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

**Customer Behaviour Analysis** is a data analytics project that explores shopping patterns using Python, SQL, PostgreSQL, and visualizes insights using Power BI. The goal is to understand customer purchasing behaviour and build actionable dashboards for business decisions.

## 🧠 About

This project analyzes customer shopping data to uncover patterns such as:

- Most popular products  
- Buying frequency trends  
- Customer segmentation  
- Sales performance over time  

You’ll find SQL queries for data preparation, a Python notebook for analysis, and a Power BI dashboard for interactive visualization.

---

## ✨ Features

- 📊 Comprehensive data exploration using Python (Pandas, Matplotlib/Seaborn)  
- 🧾 SQL queries for cleaning and preparing data  
- 📈 Interactive dashboards created with Power BI  
- 🗂️ Organized project files and reproducible workflow
